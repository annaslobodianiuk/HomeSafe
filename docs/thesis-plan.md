# Master's Thesis Plan

Recommended thesis topic:

> Intelligent system for aggregation and evaluation of rental housing for internally displaced persons based on analysis of open data sources.

## Research Object

Digital platforms for rental housing search and decision support for internally displaced persons and refugees.

## Research Subject

Methods of aggregating, normalizing, ranking and evaluating rental housing listings using open data, price comparison and fraud risk indicators.

## Goal

To develop an information system that helps internally displaced persons find reliable and affordable rental housing without intermediaries by automatically evaluating listings according to trust, value and comfort criteria.

## Main Tasks

1. Analyze existing rental housing platforms and their limitations for vulnerable users.
2. Design a data model for apartment listings, infrastructure and price history.
3. Develop a scoring approach for Trust Score, Value Score and Comfort Score.
4. Implement a React-based user interface for search and recommendation explanation.
5. Implement a Java Spring Boot backend API and PostgreSQL database.
6. Test the system on demonstration data and evaluate recommendation transparency.

## Practical Value

The system can reduce search time, improve safety, and help users compare apartments not only by price, but also by trust, infrastructure and fraud risk.

## Future Improvements

- Real integration with open housing sources and public parsers
- Map provider integration
- User authentication
- Saved searches and notifications
- Machine learning model for fraud detection
- Real market price analytics by district
